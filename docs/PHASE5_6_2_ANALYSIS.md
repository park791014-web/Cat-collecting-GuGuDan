# Phase 5.6.2 분석

- 뽑기 결과가 길어진 원인은 이미지, 등급, 이름, 중복 별, 설명, 획득 제목, 조각 안내가 모두 독립 행과 여백을 사용했기 때문이다.
- 신규 획득 제목과 추가 안내가 각각 표시돼 같은 의미가 두 번 노출됐다.
- 설명은 이전 공통 카드 스타일과 폭 제한의 영향을 받아 좁은 화면에서 단어가 부자연스럽게 분리됐다.
- 도감은 `phase561.css`의 720px 미디어 규칙이 두 열 Grid를 다시 적용했다.
- 상세 모달은 `width:100%`와 padding, 고정 이미지 크기, 긴 테마 문자열 및 flex 자식의 기본 `min-width:auto` 조합으로 가로 overflow가 생길 수 있었다.
- 수정 파일: `phase56Runtime.js`, `phase562Runtime.js`, `phase562.css`, `index.html` 및 Phase 5.6.2 문서.
