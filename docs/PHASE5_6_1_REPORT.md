# Phase 5.6.1 완료 보고

- 뽑기: 등급을 이미지 아래·이름 위로 이동하고 공통 절대 배치를 분리했다. 문 열림 완료 뒤 1초 대기, 고양이 등장 0.75초를 적용했다. 스킵 UI·이벤트·공개 디버그 함수를 제거했다.
- 미션: 진행 조건과 보상은 유지하고 제목만 `구구단 N개`, `타임어택 N회`, `모험 N회`, `정확도 100% N회` 형식으로 줄였다.
- 도감: 기본 compact 보기, 전설 우선 정렬, 52~72px 썸네일, 모바일 1열/넓은 화면 2열, 작은 중복 별을 적용했다. 첫 화면 설명은 제거하고 상세 화면의 마스터 `description`을 유지했다.
- 가로 스크롤: 4열 고정 Grid를 해제하고 카드 및 자식 최소 너비를 0으로 설정했다.
- 보존: 계정, 수집, 중복, 조각, 재화, 확률, 미션 진행과 Firebase 구조는 변경하지 않았다.
