# 냥코 구구단 대모험 V2.0.0

- 첫 시즌 `푸른 달의 러시안블루`
- 시즌 한정 고양이 6종과 전용 시각 효과 스킬
- 시즌 미션, 시즌 티켓, 시즌 팩, 최초 입장 보상
- 시즌 종료 후에도 획득 고양이 소유·장착 유지

- 4단계 온보딩과 설정 화면
