# 코인 경제 설정

- 정상 완료: 30코인
- 정답 1개: 5코인
- 정확도 100%: 30코인
- 개인 최고 기록: 50코인
- 고양이 1회 뽑기: 500코인

## 타임어택

- 정상 종료: 20코인
- 정답 1개: 3코인
- 정확도 100%: 20코인
- 개인 최고 기록: 40코인

## 구구단 대모험

- 스테이지 클리어: 플레이 보상 25코인
- 정답 1개: 5코인
- 정확도 100%: 25코인
- 기존 스테이지 최초·반복 클리어 보상은 별도로 유지

20문제 기준 일반적인 보상은 약 100~160코인으로, 약 4~5회 플레이 후 뽑기 1회를 목표로 한다. 모든 값은 `classicRewardConfig.js`와 `catDrawConfig.js`에서 중앙 관리한다.
