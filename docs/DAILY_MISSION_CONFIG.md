# 일일 미션 설정

한국 시간 `YYYY-MM-DD`가 바뀌면 진행도를 새로 생성한다.

| ID | 목표 | 보상 |
|---|---|---|
| daily_correct_30 | 누적 정답 30개 | 코인 50 |
| daily_timeattack_3 | 정상 타임어택 3회 | 코인 70, 일반 티켓 1 |
| daily_adventure_5 | 모험 클리어 5회 | 코인 80, 활성 시즌 티켓 1 |
| daily_perfect_10 | 최소 10문제, 정확도 100% 게임 10회 | 코인 120, 고급 티켓 1 |

완료와 수령 상태는 분리하며 계정·날짜별 한 번만 받을 수 있다.
