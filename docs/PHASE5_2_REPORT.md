# Phase 5.2 완료 보고서

- 도감 카드를 56~88px 중심의 자동 맞춤 썸네일로 축소했다.
- 스키마 7과 계정·게스트별 저장 키를 도입하고 공용 저장값의 1회 이전을 보호했다.
- 한국 날짜 기준 일일 미션 4종과 수동 보상·중복 잠금을 추가했다.
- 모바일 정답 버튼 높이와 글자 크기를 복구했다.
- 타임어택 랭킹에 `monthKey`와 `userId`를 저장하고 월간·전체 상위 10명 조회를 추가했다.
- 정답 발바닥·반짝임 효과와 5초 이하 타임어택 경고를 추가하고 모션 감소를 지원했다.

현재 온라인 시즌 점수 저장 계약은 존재하지 않아 시즌 명예의 전당은 허위 데이터 없이 보류했다. Firebase 실환경 쓰기·인덱스와 실제 모바일 시각 검수는 수동 확인이 필요하다.
