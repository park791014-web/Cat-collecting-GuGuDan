# 고양이 뽑기 설정

- 재화: 코인
- 비용: 500코인
- 일반 65%, 희귀 27%, 영웅 7%, 전설 1% (합계 100%)
- 후보: 기본 컬렉션이며 `available !== false`, `obtainable !== false`인 고양이만 포함
- 시즌/레거시 고양이는 제외
- 신규는 도감과 획득일을 추가하고, 중복은 기존 등급별 조각(1/2/4/8)과 중복 횟수를 지급한다.
- 코인 차감과 카드 결과를 메모리에서 계산한 뒤 한 번 저장하며 입력 잠금과 음수 방지를 적용한다.
- 기존 일반/고급/시즌 티켓 데이터는 유지하지만 현재 UI에는 표시하지 않는다.
