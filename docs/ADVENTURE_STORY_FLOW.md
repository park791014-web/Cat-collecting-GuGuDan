# 모험 스토리 실행 흐름

1. 월드맵 최초 진입: 프롤로그를 표시하고 확인 시 `prologueSeen` 저장.
2. 월드 최초 진입: 월드 인트로를 표시하고 `seenWorldIntros` 저장.
3. 스테이지 선택: 준비 화면 위에 인트로를 표시하고 `seenStageIntros` 저장. 이미 클리어한 스테이지는 생략한다.
4. 보스 도전 시작: 보스 등장 대사 확인 후에만 실제 게임과 타이머를 시작한다.
5. 최종 보스 클리어: 결과 화면 위에 clearStory를 표시하고 `seenClearStories` 저장.

모달 오류는 경고만 남기며 핵심 게임 흐름이 계속된다. Escape 또는 닫기는 시작 콜백을 실행하지 않아 준비 화면으로 안전하게 돌아간다.
